# WAB-Ansi-Logo-Make

A tiny web tool to use TDF (TheDraw font files) to create logo.

WAB-Ansi-Logo-Make is using [CODEF](https://codef.santo.fr)

## Screenshot
![logo1](README_Medias/screenshot.png)

## Usage
- Enter you name/nick/handle/text
- Choose a font
- Profit ;)
